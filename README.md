## development

```shell
$ git clone git@github.com:kamataryo/git-cloud-lesson.git
$ cd git-cloud-lesson
$ bundle install --path vendor/bundler
$ bundle exec rake start
```

## deploy

```shell
$ bundle exec rake deploy
```
